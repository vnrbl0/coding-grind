Semantic Tags in HTML

Semantic tags clearly describe their meaning in a human- and machine-readable way. They help structure the content and improve accessibility and SEO.

Examples of Semantic Tags

  • <header>: Represents introductory content or navigation links.
  • <nav>: Defines navigation links.
  • <main>: Specifies the main content of a document.
  • <article>: Represents a self-contained composition.
  • <section>: Defines sections in a document.
  • <footer>: Specifies a footer for a document or section.
