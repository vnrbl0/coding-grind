HTML Video, Audio & Media Elements Explained

Video Element (<video>):
The <video> tag is used to embed video files in a web page. You can specify the video source using the src attribute, set the width and height, and add controls for play, pause, and volume.
Example:
<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
  Your browser does not support the video tag.
</video>

Audio Element (<audio>):
The <audio> tag lets you embed sound content. Like <video>, it supports the src attribute and controls for playback.
Example:
<audio controls>
  <source src="sound.mp3" type="audio/mpeg">
  Your browser does not support the audio element.
</audio>

Other Media Elements:

  • SVG (<svg>): Used for scalable vector graphics. Example:
    <svg width="100" height="100">
      <circle cx="50" cy="50" r="40" fill="red" />
    </svg>
  • Image (<img>): Embeds images. Example:
    <img src="image.jpg" alt="Description">
  • Iframe (<iframe>): Embeds another webpage or media, such as YouTube videos. Example:
    <iframe src="https://example.com" width="300" height="200"></iframe>
