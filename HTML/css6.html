CSS Sizing Units-Compelete Guide

CSS sizing units are to define dimensions, spacing, font sized, and other measurements in web development. They fall into two main categores: Absolute Unitsand Relateive Units.

1. ABSOLUTE UNITS

Absolute units have fixed sized regardless of the parent element or viewport.

1.1 Pixels (px)

The most common unit. One pixel equals one dot on the screen.

            .box{
                width: 200px;
                height: 100px;
                font-size: 16px;
                margin: 10px;
            }

Use Cases: Borders, small margins, precise control neede

1.2 Points (pt)

Primarily used for print media 1pt=1/72 of an inch.

            @media print {
                .document {
                    width: 14pt;
                    height: 12pt;
                }
            }

Use Cases: Print styles, typography in print documents

1.3 Inches (in)

Used for print media, 1 inch equals 96 pixels.

            @media print {
                .page {
                    width: 8.5in;
                    height: 11in;
                    margin: 1in;
                }
            }

RELATIVE UNITS

Use cases:Print layouts, physical dimensions requirements

2. RELATIVE UNITS

Relative units scale based on other elements or viewport dimensions.

2.1 Percentages (%)

Relative to the parent element's dimension.

            .parent {
                width: 400px;
                padding: 200px;
            }

            .child{
                width: 50%;         /* 200px (50% of 400px) */
                height: 75%;        /* 150px (75% of 200px) */
                margin-left: 25%;   /* 100px (25% of 400px) */
            }

Use Cases: Fluid layouts, responsive designs

2.2 Em (em)

Relative to the font-size of the element itself (or parent if no font size is set)

            .parent{
                font-size: 16px;
            }

            .child{
                font-size: 1.2em; /* 1.2 * 16px = 19.2px */
                margin: 2em; /* 2 * 16px = 32px */
                padding: 1.5em; /* 1.5 * 16px = 24px */
            }

Use Cases:Scalable spacing typography, component-based design

2.3 Root EM (rem)

Always relative to the root element;s font-size (usually html)

            html{
                font-size: 16px; /* Root font size */
            }

            .header{
                font-size: 2rem;     /* 2 * 16px = 32px */
                padding: 1rem;       /* 1 * 16px = 16px */
                margin-bottom: 1rem  /* 1 * 16px = 16px */
            }

            .content{
                font-size: 1rem;     /* 1 * 16px = 16px */
                padding: 2rem;       /* 2 * 16px = 32px */
            }
Use Cases:Consistent typography, scalable layouts

3. VIEWPORT UNITS

Viewport units are relative to the size of the viewport (browser window).

3.1 Viewport Width (vw)

1vw equals 1% of the viewport width.

            .container {
                width: 80vw; /* 80% of viewport width */
                height: 50vh; /* 50% of viewport height */
                padding: 2vw; /* 2% of viewport width */
            }

Use Cases: Full-width sections, responsive designs

3.2 Viewport Height (vh)

1vh equals 1% of the viewport height.

            .hero-section {
                height: 100vh; /* Full viewport height */
                padding: 5vh; /* 5% of viewport height */
            }

            .sidebar{
                width: 20vw; /* 20% of viewport width */
                height: 100vh; /* Full viewport height */
            }

Use Cases: Full-screen sections, responsive layouts

3.3 Viewport Minimum (vmin) and Max (vmax)

  • vmin:1% of the smaller viewport dimension
  • vmax:1% of the larger viewport dimension
            .square{
                width: 50vmin;   /* 50% of smaller viewport dimension */
                height: 50vmax;  /* 50% of larger  viewport dimension */
            }

Use Cases: Full-screen sections, responsive typography, mobile-first design

4. MODERN UNITS

4.1 Character Units (ch)

Based on the widht of the "0" character in the element's font.

            .code-block{
                widht: 60ch;      /* Approximately 60 character wide */
            }

Use Cases: Typography, readable text lenghts

4.2 Ex Units (ex)

Based on the height of the "X" character (x-height).

            .subscript{
                font-size: 0.8ex;
                vertical-align: -0.2ex;
            }

Use Cases:Fine typography adjustments, mathematical expressions
