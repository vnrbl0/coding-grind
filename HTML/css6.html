Absolute Units:

  • Pixels (px) - most common for precise control
  • Points (pt) - mainly for print media
  • Physical units (in, cm ,mm) - for print layouts

Relative Units:

  • Perentage (%) - relateive eto parent element
  • Em - relative to element's font-size
  • Rem - relative to root font-size (recommended)

Viewport Units:

  • vw/vh - percentage of viewport width/height
  • vmin/vmax - based on smaller/larger viewport dimension

Modern Units:

  • ch - character width (great for readable text)
  • ex - x-height based
