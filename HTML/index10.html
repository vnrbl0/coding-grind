Definition of HTML Entities

HTML entities are special codes used in HTML to display reserved characters (like <,>, &) or characters that are not present on the keyboard (like ©, ®, €).

They usually start with an ampersand (&) and end with a semicolon (;). This is necessary because some characters are reserved in HTML and cannot be written directly (for example < is used for HTML tags).

Common HTML Entities

  • &amp; - Ampersand (&)
  • &lt; - Less than (<)< /li>
  • &gt; - Greater than (>)
  • &quot; - Double quote (")
  • &#39; - Single quote (')
  • &copy; - Copyright ©
  • &reg; - Registered ®
  • &euro; - Euro €

Input

<p>5 &lt; 10 &amp; 20 &gt; 15</p> <p>Copyright &copy; 2025 </p>

Output

5 < 10 & 20 > 15

Copyright © 2025

Using the Code Tag

The <code> tag is used to display code snippets in HTML. It is typically used to show programming code or other technical content.

<!DOCTYPE html>
<html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Code Example</title> </head> <body> <h1>Hello, World!</h1> <p>This is a simple HTML document.</p> </body> </html>

Note

When using the <code> tag, it is important to escape any HTML characters within the code snippet to prevent them from being interpreted as HTML.

For example, use &lt; for < and &gt; for >.
