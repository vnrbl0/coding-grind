Core Display Types:

  • Block: Full-width elements that start on new lines
  • Inline: Elements tht flow with text

Modern Layout Methods:

  • Flex: One-dimensiional flexible layouts with alignment control
  • grid- Two-dimensional grid system for complex layouts

Other Display Values:

  • None: Completely hides elements
  • table/table-cell/table-row: CSS table layouts
  • list-item: Custom list styling
