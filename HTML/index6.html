Difference between id and class in HTML

Feature id class
Uniqueness Must be unique within a page Can be used on multiple elements
Selector in CSS #idname .classname
Usage Identify a single element Group multiple elements
JavaScript Access getElementById() getElementsByClassName()

Examples

Using id:

    <div id="unique-element">This is a unique element</div>
This is a unique element

Using class:

    <div class="grouped-element">Element 1</div>
    <div class="grouped-element">Element 2</div>
Element 1
Element 2

Complex Examples

Combining id and class with CSS:

    <style>
      #special { color: white; background: #8e24aa; }
      .highlight { font-weight: bold; background: #c8e6c9; }
    </style>
    <p id="special" class="highlight">This paragraph uses both id and class.</p>

This paragraph uses both id and class.

JavaScript Example:

    <button id="btn">Click Me</button>
    <div class="msg">Message 1</div>
    <div class="msg">Message 2</div>
    <script>
      document.getElementById('btn').onclick = function() {
        var msgs = document.getElementsByClassName('msg');
        for (var i = 0; i < msgs.length; i++) {
          msgs[i].style.background = '#ffccbc';
        }
      }
    </script>

Message 2
